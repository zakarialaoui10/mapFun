# mapFun
# Javascript 
```javascript
console.log(
  mapFun((a) => a + 2, 1, "a", [1, 2, 3], true, {
    a: 2,
    b: 3,
    c: { d: 3, e: [1, 3, 4] },
  })
);
const sin=(...args)=>mapFun(Math.sin,...args)
```
